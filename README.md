## Instalação

Inicie o projeto com npm i na pasta onde estão os package.json

```bash
  npm i
```
    
## Deploy

Para fazer o deploy desse projeto na pasta frontend

```bash
  npm start
```

Para fazer o deploy desse projeto na pasta backend

```bash
  npm start
```
